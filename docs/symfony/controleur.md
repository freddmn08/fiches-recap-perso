## Générer un contrôleur avec la console

Cela est possible en installant le [bundle Symfony Maker](https://symfony.com/doc/current/bundles/SymfonyMakerBundle/index.html). On crée alors un contrôleur avec la commande `php bin/console make:controller MyNewController`.
